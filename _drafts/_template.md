To start a new draft from this template:
  cp _drafts/_template.md _drafts/<slug>.md

To publish a draft:
  git mv _drafts/<slug>.md _posts/YYYY-MM-DD-<slug>.md
  git commit -m "Add post: <title>"
  git push

Notes:
- Don't start your body with "# Title" — the layout already renders the
  page title as an <h1>. Use ## for in-post section headings.
- The date in front matter and the date in the filename should match.
- Multi-line bullets work via lazy continuation — no need to indent.
